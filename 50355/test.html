Week Three:
Selections

This week we will go over how to select things in Photoshop. While working in Photoshop one of the key things to know is how you can select an object. Without knowing how to select objects or create different types of selections, creating your artwork or graphics is very difficult.

So in this module we will go over the many different selection tools inside of Photoshop and then you will create a fun piece of artwork. Please read the selections overview, it is quite long as there are multiple ways to create selections in Photoshop. Also view the selections demo and complete the assignment, which is due
Saturday, January 30, 2016 at 11:59pm EST.

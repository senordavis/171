Week Thirteen:
Web Design

Last week you took the time to work on designing a desktop version of the website. A number of years ago when creating web designs that would suffice.

But now we live in a world where mobile devices like phones and tablets connect us to the internet in many different ways. And with those different ways comes different screen sizes and different resolutions.

This week you will need to take your desktop designs and reduce them to mobile versions.

In this module we will go over how to create a web design. Complete the assignment, which is due Saturday, April 16, 2016 at 11:59pm EST.

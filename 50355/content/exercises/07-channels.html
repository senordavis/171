Objectives

After successful completion of this assignment, the student will be able to:

  • Demonstrate an understanding of Channels
  • Create a detailed complex selection using Channels
  • Identify shadow, highlight and neutral-gray midtone points in an image
  • Use a Layer Mask to cut-out an element from an image

Assignment Specifications

For this exercise you will be using channels to make selections on two different Photographs.

The first image, lotus.jpg, you will use channels to create a saved selection (Alpha Channels) of the flower. Use this selection to create a Layer Mask around the flyer and then place it on a different background.

For the second image, I've added a few versions of the photo so that you may choose one of them to create a selection using channels. Create a Layer mask around the woman in the photograph and place her on a different background.

Save your files with appropriate names, following the file naming convention:


  yourname-channels-1.psd 

  yourname-channels-2.psd

Once completed, upload your images to the Channels Dropbox.

Please make sure to upload your files as PSDs. If your file is to large, please upload your PSD to the LCC Google Drive and share a link to the file with me. Then upload a JPG to the D2L Dropbox for this assignment.

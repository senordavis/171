Objectives

After successful completion of this assignment, the student will be able to:

  • Color correct an image that demonstrates a color bias
  • Utilize Curves in conjunction with the Info Palette to identify color bias in an image
  • Identify shadow, highlight and neutral-gray midtone points in an image
  • Utilize black point, white point and midtone point in Curves
  • Manipulate individual color channel Curves to get accurate color correction

Assignment Specifications

From the module in D2L download the provided images. These images exhibit some kind of color bias. Your final Photoshop files will only contain layers for the photographic image and the Curves adjustment layer.

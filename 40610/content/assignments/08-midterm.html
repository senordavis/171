Objectives

After successful completion of this assignment, the student will be able to:

  • Create new Action Sets
  • Record new Actions into their Sets
  • Insert Modal Control Stops in their recorded Actions
  • Export saved Action Sets as individual files

Assignment Specifications

Create three different actions as described below. To begin making your actions, create a new set in the Actions Palette by clicking on the New Set icon (it looks like a manila folder just like the New Group icon in the Layers Palette). Please label your set as yourname-actions.

Save your file with appropriate names, following the file naming convention:


  yourname-midterm.atn

Once completed, upload your files to Fantastic Beasts Midterm Assignment Submission Folder.

notification_important

Please make sure to upload your files as PSDs. If it exceeds the file size allowed by D2L, please upload the PSD to the LCC Google Drive and share a link to it with me. Then upload a JPG to the D2L Assignment Submission Folder for this assignment.

If you are having problems, please view the upload guide video, or email me.

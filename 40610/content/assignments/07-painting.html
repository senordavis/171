Project Description

Given the name, Photoshop has traditionally focused on manipulating and refining photos. The tools exist, however, to also take those photos from realistic imagery to more painterly presentations.

As a professional digital artist, you want to respect both the original images and the pixels used in the digital file.

Photoshop brushes have realistic textures and bristles. They resemble their real-world counterparts allowing you to create an effect like that of a real drawing or painting.

Begin this project by finding an image that you would like to add a painterly effect to, and use that image as a guide.

Project Specifications

To satisfy the project requirements, you must submit one finished high-resolution image at least 1500 pixels by 2000 pixels.

You will also need to submit your original un-retouched image. You can include the original image as a separate layer in the final PSD.

Save your image with the appropriate name, following the file naming convention:


  yourname-painting.psd

Once completed, upload your images to the Painting Assignment Submission Folder.

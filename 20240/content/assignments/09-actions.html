Objectives

After successful completion of this assignment, the student will be able to:

  • Create new Action Sets
  • Record new Actions into their Sets
  • Insert Modal Control Stops in their recorded Actions
  • Export saved Action Sets as individual files

Assignment Specifications

Create three different actions as described below. To begin making your actions, create a new set in the Actions Palette by clicking on the New Set icon (it looks like a manila folder just like the New Group icon in the Layers Palette). Please label your set as yourname-actions.

Next, create a new action in the Actions Palette by clicking on the New Action icon (it looks like a folded post-it note just like the New Layer icon in the Layers Palette). Please label your individual actions as yourname-action-# (replace # with the correct number).

Begin recording your action by clicking on the Record Button. Once finished recording, press the Stop Button. Repeat for all actions.

To save your actions for grading, select the Set in your Actions Palette (you cannot save individual actions, only the set). Select the More Options button and pick the “save actions...” menu item. Your file will be a the set name with an .atn extension, this is what you will turn in for grading.

Please create the following actions:

Action 1

  • Change the Image Size to 5” x 7” at 150 dpi
  • Adjust the Levels output (bottom slider) to 20 & 240
  • Apply the Unsharp Mask with the following settings:
    • Amount: 125%
    • Radius: 1 pixel
    • Threshold: 2
  • Convert the CMYK color mode
  • Save image as a TIFF to Desktop

Action 2

  • Rotate image 90° counter-clockwise
  • Change Image Size to 300 dpi
  • Convert image to Grayscale color mode
  • Apply the Unsharp Mask with the following settings:
    • Amount: 105%
    • Radius: 1 pixel
    • Threshold: 1

(please include modal control stops at the Image Size and the Unsharp Mask dialogs)

Action 3

Create your own action to make alterations to an image. There should be at least four steps to your Action and at least one modal control stop. Otherwise be as creative and fun as you want.

Save your file with appropriate names, following the file naming convention:


  yourname-actions.atn

Once completed, upload your files to Actions Dropbox.

Please make sure to upload your files as PSDs. If your file is to large, please upload your PSD to the LCC Google Drive and share a link to the file with me. Then upload a JPG to the D2L Dropbox for this assignment.

If you are having problems please view the Upload guide or email me.

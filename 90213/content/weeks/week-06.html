Assignment Ten:
Cross Section

As a kid having x-ray vision was always one of the top super powers people always wanted. Now using Photoshop you can create an image that gives you that ability. By using many of the techniques you have learned this semester you can create an interesting cross section of an image.

In this module we will go over how to create a cross section effect. View the cross section demo and complete the assignment, which is due Saturday, July 15, 2017 at 11:59pm EST.

Assignment Eleven:
Video Clips

Sometimes you may need to quickly edit a video or crop it to a specific length. There are many video editing programs that exist, but it can take a long time to learn how to use them. While Photoshop's name would imply that it is strictly for photos, this module will show you that it can do much more.

In this module we will go over how to create a short video clip/effect. View the video clip demo and complete the assignment, which is due Saturday, July 15, 2017 at 11:59pm EST.
